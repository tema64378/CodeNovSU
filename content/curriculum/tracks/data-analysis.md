# Data Analysis Track

Уровни:

- `Beginner`: таблицы и базовые статистики
- `Basic`: группировка данных и визуальные выводы
- `Intermediate`: нормализация и one-hot encoding
- `Advanced`: аналитический мини-проект по воронке

Итоговый capstone:

- расчёт конверсии и краткая продуктовая интерпретация
