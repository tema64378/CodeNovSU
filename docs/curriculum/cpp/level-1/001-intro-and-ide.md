# C++ · Beginner

## Урок 001: Введение и среда разработки

### Зачем этот урок
Понять цепочку **исходник → компиляция → линковка → запуск**, собрать минимальный проект и читать ошибки компилятора.

### Главное
- **Точка входа** — функция `main`.
- **Сборка**: препроцессор → компиляция объектных файлов → линковка в исполняемый файл. Ошибки **компиляции** (синтаксис, типы) и **времени выполнения** (логика, данные) — разные.
- **Компиляторы**: GCC, Clang, MSVC. Явно задавайте стандарт (например `-std=c++17`).
- **Минимальный стек**: компилятор + редактор (или IDE) + терминал.

### Пример
```cpp
#include <iostream>

int main() {
    std::cout << "Hello, C++!" << std::endl;
    return 0;
}
```

**Терминал (пример):**
```bash
g++ -std=c++17 main.cpp -o app
./app
```

### Практика
1. Проверить версию компилятора (`g++ --version` или аналог).
2. Создать `main.cpp`, вывести строку, собрать и запустить.
3. Сломать код намеренно и прочитать сообщение компилятора.

### Контроль
